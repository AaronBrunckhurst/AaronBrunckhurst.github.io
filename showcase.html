Pinpoint AVL

As co‑founder and Lead Software Engineer, I led software‑in‑the‑loop testing and developed much of the embedded firmware for the Pinpoint AVL system. The platform was deployed on 19 campus buses and provided real‑time location tracking, dispatching, analytics, and operational insights.



Slugbotics ArmTeam

On ArmTeam, I led the integration of visual object identification and LLM‑based command execution. This included real‑time computer vision pipelines, robotic control logic, and natural‑language driven task execution for a 3D‑printed robotic arm.

Confab Agora

Confab Agora is a communications and game distribution platform created in C# and WPF. It supports real-time voice and text chat, and served as the store and launcher for games produced by Callen Games.

Wizards Duel

Wizards Duel is a turn-based pattern matching combat game where two wizards face off by casting powerful spells. Inspired by the classic memory game *Simon*.
